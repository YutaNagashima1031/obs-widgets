# OBS Stream Widgets

このリポジトリは、OBS StudioやStreamlabs Desktopなどの配信ソフトで使用できる、カスタムメイドのローカルHTMLウィジェット集です。
ブラウザソースとして読み込むだけで、配信画面にそのまま配置できるように背景の透過処理などを施しています。

## **収録ウィジェット**

懐かしのレトロなPCウィンドウをモチーフにしたデジタル時計ウィジェットです。実際の配信画面に重ねると、背景が綺麗に透過されて表示されます。

* **主な機能と特徴**
  * OBS上で自動的に背景が透過される専用設計
  * Windows XP風の青いグラデーションタイトルバーと閉じるボタンの装飾
  * 視認性が高くスタイリッシュな太字フォント（Montserrat）を採用
  * 1秒ごとに点滅するコロンと、曜日・日付の自動取得機能
  * 配信中をアピールする「LIVE」バッジデザイン

### 2. すりガラス風 天気予報ウィジェット (`weather/weather_widget.html`)

![天気予報ウィジェットのプレビュー]<img width="109" height="126" alt="obs - コピー" src="https://github.com/user-attachments/assets/834d88dd-9b00-4db9-9a20-3805bd25fe47" />

指定した複数都市の天気を自動でループ表示する、美しくモダンな天気予報ウィジェットです。背後のゲーム映像やカメラ映像がほんのりと透ける仕様になっています。

* **主な機能と特徴**
  * Open-Meteo APIを利用した完全自動の天気データ取得（1時間ごとにバックグラウンド更新）
  * 日本全国7都市（札幌、仙台、新潟、東京、名古屋、大阪、福岡）を5秒間隔でフェードインしながら順番に表示
  * 天気（絵文字アイコン）、最高気温（赤）、最低気温（青）、降水確率をひと目で確認可能
  * OBSの背景映像と綺麗に馴染む、乳白色のすりガラス調（backdrop-filter）デザイン

## **OBS / Streamlabs での導入方法**

これらのウィジェットを配信画面に表示させるための手順です。

1. このリポジトリ全体を自分のPCにダウンロード（またはClone）します。
2. OBS（またはStreamlabs）を開き、「ソース」の「+」ボタンから**「ブラウザ」**を追加します。
3. 新規作成の画面が出たら、**「ローカルファイル」**のチェックボックスをオンにします。
4. 「参照」ボタンを押し、ダウンロードしたフォルダの中から表示させたいHTMLファイルを選択します。
5. ソースのプロパティ画面で、以下のサイズを目安に設定してください。
   * **🕰️ 時計ウィジェットの場合**
     * 幅 (Width): `300`
     * 高さ (Height): `300`
   * **⛅ 天気ウィジェットの場合**
     * 幅 (Width): `200`
     * 高さ (Height): `250`
   *(※CSSの影やアニメーションが途切れないよう、実際のサイズより少し余裕を持たせた数値を推奨しています。お好みで調整してください)*
6. 「OK」を押すと、背景が透過された状態で配信画面に表示されます。ソースをドラッグして好きな位置に配置してください。

## **カスタマイズについて**

HTMLファイル内のコードを編集することで、お好みのデザインや機能にアレンジすることが可能です。

* **色の変更**: `<style>` タグ内のカラーコード（例：`#0058EE` や `rgba(255, 255, 255, 0.5)`）を変更してください。
* **表示する都市の変更**: 天気ウィジェットの `<script>` タグ内にある `const cities = [...]` のリストを書き換えることで、地名や緯度・経度を自由に追加・変更できます。
